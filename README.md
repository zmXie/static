# Static Tools

一个简单的静态 HTML 工具集，部署在 Vercel 上。

## 工具列表

1.  **今天吃什么** (`what-to-eat.html`)：解决“世界性难题”——今天吃什么。支持转盘旋转和自定义配置。
2.  **文档转换助手** (`word2html.html`)：将 Word (.docx) 文件转换为标准的、干净的 HTML 内容。

## 技术栈

- 纯 HTML / CSS / JavaScript
- Vue 3 (通过 CDN 引入)
- Element Plus (通过 CDN 引入)
- Mammoth.js (Word 解析器)

## 部署

项目通过 Vercel 自动化部署。每当推送代码到 GitHub 仓库时，Vercel 会自动更新。
